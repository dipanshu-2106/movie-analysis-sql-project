# 🎬 Movie Analysis SQL Project (Advanced Database Design)

## 📌 Overview
This project is a complete Movie Analysis database built using SQL Server. It includes multiple relational tables and advanced SQL queries to extract meaningful insights from movie data such as ratings, revenue, actors, directors, genres, and box office performance.

The project demonstrates real-world database design and data analysis techniques using SQL.

---

## 🗂️ Database Structure

The project consists of multiple related tables:

- 🎭 genres
- 🎬 movies
- 🎥 directors
- 🎭 actors
- 🎬 movie_cast (Many-to-Many relationship)
- ⭐ reviews
- 💰 box_office

---

## 🧠 Key Features

✔ Relational database design (PK-FK relationships)  
✔ Many-to-Many mapping (movies ↔ actors)  
✔ Data integrity using constraints  
✔ Indexing for performance optimization  
✔ Real-world movie dataset simulation

## 📊 SQL Analysis Performed

This project answers important analytical questions like:

### 🎯 Movie Performance

- Most successful genres

### 🎬 Director Analysis
- Most active directors

### 🎭 Actor Insights
- Most active actors

### 💰 Box Office Analysis
- Weekly box office trends
- Country-wise collections

### ⭐ Review Analysis
- Most helpful reviews
- Average rating per movie
